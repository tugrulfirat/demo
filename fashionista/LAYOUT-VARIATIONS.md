# Layout Variations Across 3 Home Pages

## 🎯 Strategy
Each page will have completely different layouts for the same sections, giving you maximum variety.

---

## 📄 **index.html** - GRID FOCUSED

### Hero: Gradient (current)
### Featured Collection: 4-column grid
### Categories: 4-column grid (current)
### New Arrivals: 4-column grid (current)
### Limited Edition: Full-width centered
### Bestsellers: 4-column grid with ratings
### Editorial: 50/50 split
### Shop by Style: Masonry grid (1 large + 4 small)
### Newsletter: Centered
### Footer: 4-column grid

**Theme:** Clean, organized, traditional e-commerce

---

## 📄 **index-split.html** - HORIZONTAL SCROLL FOCUSED

### Hero: Split (50/50 with image + content)
### Featured Collection: **Horizontal scroll carousel**
### Categories: **2 large cards (horizontal)**
### New Arrivals: **Horizontal scroll with snap**
### Limited Edition: **Full-width with side-by-side content**
### Bestsellers: **Horizontal scroll**
### Editorial: **Reversed split (content left, image right)**
### Shop by Style: **Horizontal scroll gallery**
### Newsletter: **Inline with background image**
### Footer: **Horizontal sections**

**Theme:** Modern, swipeable, mobile-first

---

## 📄 **index-asymmetric.html** - MASONRY/ASYMMETRIC FOCUSED

### Hero: Asymmetric (45/55 with products)
### Featured Collection: **Staggered grid (different heights)**
### Categories: **Asymmetric (1 large, 3 small)**
### New Arrivals: **Masonry layout (Pinterest-style)**
### Limited Edition: **Diagonal split with overlay**
### Bestsellers: **Showcase layout (1 large + 3 sidebar)**
### Editorial: **Full-width with floating content box**
### Shop by Style: **Bento box grid (varied sizes)**
### Newsletter: **Offset card on colored background**
### Footer: **Asymmetric columns**

**Theme:** Editorial, magazine-style, dynamic

---

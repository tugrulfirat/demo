# Fashionista Home Page Versions

## 📄 index.html - **Classic Gradient Hero**

### Hero Section:
- Full-viewport gradient background (blush to lavender)
- Centered text with "Timeless Elegance"
- Single CTA button
- Scroll indicator at bottom
- **Best for:** Minimalist, elegant first impression

### Section Order:
1. Hero (Gradient)
2. Featured Collection (4 products carousel)
3. Category Grid (Women, Men, Accessories, Sale)
4. New Arrivals (8 products)
5. Limited Edition Highlight (Pink background)
6. Bestsellers (4 products with ratings)
7. Editorial Section (Image + Text split)
8. Shop by Style (5 style categories)
9. Newsletter
10. Footer

## 🎨 How to Customize Further

### Different Product Images Per Page:
Each version can feature different products in their sections to create unique experiences:

**index.html** - Focus on Spring Collection
**index-split.html** - Focus on Bestsellers & Classics
**index-asymmetric.html** - Focus on New Arrivals & Trending

### Different Color Schemes:
- **index.html**: Blush + Lavender gradient
- **index-split.html**: Cream + Sage backgrounds
- **index-asymmetric.html**: Rose + Blush gradient

### Different Section Orders:
Rearrange sections to test different user journeys and see which converts better.

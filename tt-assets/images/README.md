# TrendyThread Images Directory

## Quick Start

This directory contains all images for the TrendyThread e-commerce site.

### Directory Structure

```
images/
├── products/     → Product images (3:4 aspect ratio required)
├── hero/         → Homepage hero banners
├── categories/   → Category banner images
└── blog/         → Blog post images
```

## Important: Image Requirements

### Product Images MUST be 3:4 Aspect Ratio
- ✅ 600×800px
- ✅ 900×1200px (recommended)
- ✅ 1200×1600px
- ✅ 1500×2000px

### File Format
- JPG format
- Optimized for web (< 300KB per image)
- sRGB color profile

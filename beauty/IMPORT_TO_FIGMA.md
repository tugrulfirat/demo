## Method 1: Drag & Drop (Fastest ⚡)

### In Figma:
1. **Open your Figma file** (or create new: figma.com)
2. **Drag screenshots folder directly into canvas**

## Method 2: Using File → Import

### In Figma:
1. **Go to:** File → Import
2. **Select all .png files** from `./screenshots/` folder
3. **Click Open**
4. **Images are imported as frames**

## Troubleshooting

**Q: Screenshots are too small?**
- Select frame → Click resize corner → Scale up
- Or double-click image to edit size

**Q: Imported as images instead of frames?**
- Right-click → Create component
- Or: Object → Flatten selection

**Q: Want to edit the images?**
- Use File → Place image → Update
- Or export from screenshots folder and re-import

**Q: How to align multiple images?**
- Select all frames (Shift+click)
- Use Align tools in right panel
